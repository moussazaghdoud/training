# Rainbow Training Platform — Certification & Assessment Framework

> Phase 2 Deliverable | Certification Tiers, Assessment Strategy & Badge System

---

## PART A: CERTIFICATION ARCHITECTURE

### Certification Tiers

The platform offers a **3-tier certification structure** per persona track:

| Tier | Badge | Requirements | Validity |
|---|---|---|---|
| **Certified** | Silver | Complete L1 + L2, pass knowledge assessment (70%) | 2 years |
| **Expert** | Gold | Complete L1-L3, pass expert assessment (80%) | 2 years |
| **Champion** | Platinum | Complete L1-L4, pass final certification (85%) + practical | 1 year (renewable) |

### Certifications by Track

| Track | Certified (Silver) | Expert (Gold) | Champion (Platinum) |
|---|---|---|---|
| Support | Rainbow Certified Support Agent | Rainbow Support Expert | Rainbow Support Champion |
| Customer Success | Rainbow Certified CS Associate | Rainbow CS Expert | Rainbow CS Champion |
| Executives | Rainbow Certified Executive | Rainbow Executive Strategist | Rainbow Executive Champion |
| Partners | Rainbow Certified Partner | Rainbow Partner Expert | Rainbow Partner Champion |
| Developers | Rainbow Certified Developer | Rainbow Developer Expert | Rainbow Developer Champion |

## PART B: ASSESSMENT TYPES & CRITERIA

### 1. Knowledge Checks (QIZ) — Formative
| Attribute | Specification |
|---|---|
| Purpose | Reinforce learning within modules; no certification impact |
| Format | Multiple choice, matching, drag-and-drop, true/false |
| Questions per check | 5-10 |
| Passing score | 60% (informational — can retry immediately) |
| Attempts | Unlimited |
| Time limit | None |

### 2. Level Assessments — Summative
| Attribute | Specification |
|---|---|
| Purpose | Gate to next level; contributes to certification |
| Format | Multiple choice, scenario-based, matching, ordering |
| Questions | 20-30 per assessment |
| Passing score | 70% (Certified), 80% (Expert), 85% (Champion) |
| Attempts | 3 per 30-day period |
| Time limit | 45 minutes |
| Proctoring | Optional (recommended for Champion tier) |

### 3. Practical Assessments — Champion Only
| Attribute | Specification |
|---|---|
| Purpose | Validate real-world competency |
| Format | Scenario simulation, live demo, or project submission |
| Duration | 60-90 minutes |
| Evaluation | Rubric-based scoring by certified evaluator |
| Passing score | 85% on rubric |
| Attempts | 2 per 60-day period |

## PART E: RECERTIFICATION & MAINTENANCE

| Tier | Validity | Recertification Method |
|---|---|---|
| Silver (Certified) | 2 years | Pass updated assessment (70%) OR complete delta modules |
| Gold (Expert) | 2 years | Pass updated assessment (80%) OR complete delta modules + 1 case study |
| Platinum (Champion) | 1 year | Attend annual champion summit + pass updated assessment + contribute 1 KB article or case study |

### Delta Modules
When Rainbow releases significant new features, **delta modules** are published covering:
- New feature overview (15-30 min)
- Updated assessment questions
- Updated reference cards

Certified holders are notified and given 90 days to complete delta modules to maintain their certification currency.

## PART F: PLATFORM REQUIREMENTS FOR ASSESSMENT

| Requirement | Specification |
|---|---|
| Question bank size | 3x the assessment length (to enable randomization) |
| Randomization | Questions and answer order randomized per attempt |
| Item analysis | Track difficulty, discrimination, and distractor effectiveness |
| Accessibility | WCAG 2.1 AA compliant assessment interface |
| Language support | English (primary), French, Spanish, German (Phase 2+) |
| Reporting | Per-learner score breakdown by domain, per-cohort analytics |
| Anti-cheating | Timer, randomization, limited attempts, optional proctoring |
| Mobile support | Assessments must be completable on tablet (phone not required) |
